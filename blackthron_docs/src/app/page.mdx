# Blackthorn API

Blackthorn is a pre-signature transaction security layer for Solana. Before a user signs, Blackthorn simulates the transaction, runs 7 independent risk detectors, evaluates your policy rules, and returns a clear `safe: true/false` verdict with full findings. {{ className: 'lead' }}

## How it works {{ anchor: false }}

Every request to `/v1/analyze` goes through the same pipeline: decode the transaction, resolve Address Lookup Tables, fetch live account states from RPC, simulate execution without broadcasting, then run 7 specialized detectors in parallel. The policy engine evaluates your configured rules against the findings and returns a final verdict — all before any signature is requested. {{ className: 'lead' }}
