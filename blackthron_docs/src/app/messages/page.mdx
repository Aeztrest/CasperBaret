# Stream Analysis

The streaming endpoint runs the same analysis pipeline as `/v1/analyze` but emits results incrementally via Server-Sent Events (SSE). Each detector sends its findings as it completes — no waiting for the full analysis to finish. {{ className: 'lead' }}

---

## Stream a transaction {{ tag: 'POST', label: '/v1/analyze/stream', id: 'stream-a-transaction' }}

<Row>
  <Col>

    ### Required attributes

    <Properties>
      <Property name="transaction" type="string">
        Base64-encoded serialized Solana `VersionedTransaction`.
      </Property>
      <Property name="cluster" type="string">
        Solana cluster: `mainnet-beta` or `devnet`.
      </Property>
    </Properties>

    ### Optional attributes

    <Properties>
      <Property name="policy" type="object">
        Inline policy rules. See [Policy Engine](/webhooks).
      </Property>
      <Property name="policyProfile" type="string">
        Pre-configured policy profile: `strict`, `defi-permissive`, or `monitor-only`.
      </Property>
    </Properties>

    The response uses `Content-Type: text/event-stream`. Each SSE message has a `event` name and `data` payload. The stream closes after the `complete` event.

  </Col>
  <Col sticky>

    <CodeGroup tag="POST" label="/v1/analyze/stream">

    ```js
    const response = await fetch('https://api.blackthorn.io/v1/analyze/stream', {
      method: 'POST',
      headers: {
        'X-API-Key': process.env.BLACKTHORN_API_KEY,
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({
        transaction: serializedTxBase64,
        cluster: 'mainnet-beta',
      }),
    })

    const reader = response.body.getReader()
    const decoder = new TextDecoder()

    while (true) {
      const { done, value } = await reader.read()
      if (done) break

      const text = decoder.decode(value)
      const lines = text.split('\n')

      for (const line of lines) {
        if (line.startsWith('data: ')) {
          const event = JSON.parse(line.slice(6))
          console.log(event.type, event.data)
        }
      }
    }
    ```

    ```python
    import httpx

    with httpx.stream(
        'POST',
        'https://api.blackthorn.io/v1/analyze/stream',
        headers={'X-API-Key': API_KEY},
        json={'transaction': tx_base64, 'cluster': 'mainnet-beta'},
    ) as r:
        for line in r.iter_lines():
            if line.startswith('data: '):
                event = json.loads(line[6:])
                print(event['type'], event['data'])
    ```

## Event types {{ id: 'event-types' }}

<Properties>
  <Property name="simulation" type="event">
    Emitted after the RPC simulation completes. Contains the simulation status and balance changes.
  </Property>
  <Property name="detector" type="event">
    Emitted for each detector as it completes. The `data.name` field identifies which detector (`program`, `cpi`, `reputation`, `compute`, `delta`, `token2022`). Contains findings for that detector.
  </Property>
  <Property name="policy" type="event">
    Emitted after all detectors complete and the policy engine has evaluated the findings. Contains the block decision and matched rule.
  </Property>
  <Property name="complete" type="event">
    Final event. Contains the full analysis object identical to a `/v1/analyze` response. The stream closes after this event.
  </Property>
  <Property name="error" type="event">
    Emitted if the analysis fails. Contains an error code and message. The stream closes after this event.
  </Property>
</Properties>

```json {{ title: 'Example SSE stream' }}
event: simulation
data: {"status":"success","balanceChanges":[{"account":"7xKX...","solDelta":-0.000005}]}

event: detector
data: {"name":"program","findings":[]}

event: detector
data: {"name":"delta","findings":[{"code":"UNLIMITED_APPROVAL","severity":"high"}]}

event: policy
data: {"blocked":true,"rule":"blockApprovalChanges","reason":"Unlimited token approval detected"}

event: complete
data: {"safe":false,"riskLevel":"high","verdict":"Transaction blocked: unlimited token approval.","durationMs":301}
```
